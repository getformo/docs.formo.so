---
title: "Get KPIs"
description: "Time-series traffic KPIs (visitors, pageviews, bounce rate, session duration), with optional dimension breakdown and previous-period comparison."
openapi: "GET /v0/kpis"
---

Returns the same KPI series that powers the Formo dashboard's overview chart. Here `visitors` is the per-day session count (the overview's **Sessions** card), while `unique_users` (and `unique_users_current` / `unique_users_previous`) are unique anonymous visitors (the overview's **Visitors** card) ÔÇö returned only when `include_previous_period=true` and no `group_by` is set.

Use `group_by` to break results down by `referrer`, `location`, `device`, `browser`, `os`, or any UTM dimension. Set `include_previous_period=true` to also receive the equivalent prior window for week-over-week comparisons.
