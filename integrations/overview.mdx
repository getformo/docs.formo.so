## Wallets

Formo works out of the box with major wallets.

| Name                     | Supported |
|:------------------------|:----------|
| Metamask                 | ✅        |
| Rainbow                  | ✅        |
| Phantom                  | ✅        |
| Rabby                    | ✅        |
| EIP-1193 wallets        | ✅        |

## Embedded Wallets

Formo supports embedded wallet providers that create wallets using email, social login, or passkeys.

| Name                     | Supported |
|:------------------------|:----------|
| Turnkey                  | ✅        |
| Privy                    | ✅        |
| Dynamic                  | ✅        |
| Porto                    | ✅        |
| Reown                    | ✅        |
| Thirdweb                 | ✅        |

> Don't see your wallet? [Let us know](https://formo.so/support).

## BI Tools

Formo is compatible with Business Intelligence (BI) tools through our HTTP interface, enabling you to connect popular BI tools, SQL clients, and data visualization platforms directly to Formo. [Learn more](/data/bi).
