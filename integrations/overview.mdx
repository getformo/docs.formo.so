## Wallets

Formo works out of the box with major wallets.

| Name                     | Supported |
|:------------------------|:----------|
| Metamask                 | ✅        |
| Rainbow                  | ✅        |
| Phantom                  | ✅        |
| Rabby                    | ✅        |
| Other EVM wallets        | ✅        |

Formo supports embedded wallets.

| Name                     | Supported |
|:------------------------|:----------|
| Privy                    | ✅        |
| Dynamic                  | ✅        |
| Porto                    | ✅        |
| Reown                    | ✅        |
| Thirdweb                 | ✅        |
