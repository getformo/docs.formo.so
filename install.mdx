Pick your stack and add the code below to your codebase.

<AccordionGroup>
  <Accordion icon="code" title="Browser">
    
    <Info>
      **Prerequisite**: Please find your project API key in the Formo dashboard.
    </Info>

    Install this script in the `<head>` tag of your website:

    <CodeGroup>

      ```bash html
      <script>
        const script = document.createElement('script');
        const apiKey = '<your-project-api-key>';

        script.src = 'https://unpkg.com/@formo/analytics';
        script.onload = function () {
          FormoAnalytics.init(apiKey)
            .then((sdkInstance) => {
              window.formo = sdkInstance;
              window.formo.page();
            })
            .catch((error) => {
              console.error('Error initializing FormoAnalytics:', error);
            });
        };
        document.head.appendChild(script);
      </script>
      ```
