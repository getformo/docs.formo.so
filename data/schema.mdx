Formo’s extensive data catalog provides detailed insights on events, users, and wallet profiles across chains. 

- **Events** and **Users** are first-party data. Access is restricted to each workspace. 
- **Wallet Profiles** include public onchain data (net worth, apps, tokens, chains, labels, etc.). Globally accessible across all workspaces.

<img src="/images/explore.png" alt="Data schema" />

Sign in and go to the [Explorer](/features/product-analytics/explore) to view the Formo data schema.

Query data with the [Query API](/api-reference/overview#query-api) and [Profiles API](/api-reference/overview#profiles-api)., explore with the [BI integration](/data/bi), and export data with [Data Sync](/data/data-sync).
