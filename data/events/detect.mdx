---
title: 'Detect event'
description: "Reference for the detect event that identifies a visitor's installed wallet provider, capturing the wallet name and RDNS before a connection is made."
---

The `detect` event lets you identify a visitor's wallet name and rdns.

## Sample Payload

Here’s the payload of a typical call with most common fields removed:

```json
{
    "type": "detect",   
    "properties": {
        "provider_name": "MetaMask",
        "rdns": "io.metamask"
    }
}
```
