## Events API

- Send individual or batches of events via `events.formo.so` to Formo.
- High-throughput streaming ingestion with an easy-to-use HTTP API.
- Supports 1000 requests/s and 20MB/s.

## Profiles API

You can use the Profiles API for real-time personalization. Once you fetch a profile, you can use them in your app for real-time personalization.

- Fetch wallet profile properties including net worth and wallet labels
- Segment and target users based on location, device, and referrer
- Real-time activation API for personalized user experiences

## Query API

- Connect BI tools and SQL clients via [BI integration](/data/bi).
- Query and filter raw analytics events, materialized views, and metrics.
- Return the activity feed / event stream for an anonymous visitor or wallet.
- Export data periodically to your data warehouse.

## Boards & Charts API

- Create and manage custom dashboards and charts.
- Add, update, and remove charts with SQL-based queries and visualization options (bar, line, pie, funnel, stacked, retention, and more).
- Requires `boards:read` / `boards:write` scopes. Available on Scale and Enterprise plans.

## Segments API

- Create and manage user segments based on filter conditions.
- Segment users by transaction count, device, browser, and other attributes.
- Requires `segments:read` / `segments:write` scopes. Available on Scale and Enterprise plans.

## Contracts API

- Monitor blockchain smart contracts and their events.
- Add contracts by chain ID and address, with ABI and event definitions.
- Requires `contracts:read` / `contracts:write` scopes. Available on Scale and Enterprise plans.

## Alerts API

- Create, update, and manage project alerts programmatically.
- Configure alert conditions and notification channels (email, Slack, webhook).
- Toggle alerts active/inactive.
- Requires `alerts:read` / `alerts:write` scopes. Available on Scale and Enterprise plans.

## FAQ

<AccordionGroup>
  <Accordion title="What are the rate limits for the Formo API?">
    API rate limits depend on your plan. Check the [authentication docs](/api-reference/authentication) for details on rate limits and API key scopes.
  </Accordion>
  <Accordion title="Can I use the Profiles API for real-time app personalization?">
    Yes. The [Profiles API](/api-reference/profile) returns wallet profile data including holdings, net worth, lifecycle stage, and labels. It also returns your first-party data (if the user has visited your app) such as country, device, browser, OS, and more. You can use this to personalize your app experience based on a user's onchain activity.
  </Accordion>
  <Accordion title="Can I connect BI tools like Metabase or Looker to Formo?">
    Yes. Formo provides [BI tool integration](/data/bi) via BI Read tokens that connect to the Formo data warehouse. You can connect Metabase, Grafana, and other SQL-based tools to query your analytics data.
  </Accordion>
  <Accordion title="What are the premium APIs (Alerts, Boards, Contracts, etc.)?">
    These are APIs available on Scale and Enterprise plans. They let you programmatically manage [alerts](/api-reference/alerts/list), [dashboards and charts](/api-reference/boards/list), [contracts](/api-reference/contracts/list), [segments](/api-reference/segments/list), and [import wallets](/api-reference/import) using Workspace API Keys with the appropriate scopes. For AI-powered analytics, use the [Formo MCP server](/features/mcp/overview) to integrate with tools like Claude and Cursor.
  </Accordion>
</AccordionGroup>
