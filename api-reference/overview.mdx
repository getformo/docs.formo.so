## Events API

- High-throughput streaming ingestion with an easy-to-use HTTP API.
- Send individual or batches of events to Formo.
- The Events API supports 1000 requests/second and 20MB/s.

## SQL Query API

- Query and filter raw analytics events, materialized views, and metrics.
- Return an activity/event stream for a user.
- Export analytics data periodically to Kafka or AWS S3.

## Wallet Profiles API

- Fetch wallet profile properties including net worth and wallet labels
- Detect if a user is a bot with wallet reputation scores
- Real-time activation API that unifies wallet profile data with in-app wallet messaging

## Forms API

- Use the Forms API to trigger [webhooks](/api-reference/webhooks) based on form responses. 
- Trigger thousands of other apps such as Google Sheets, and more whenever you get a new form response on Formo.
