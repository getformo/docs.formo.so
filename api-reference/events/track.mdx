---
title: 'Track Event'
description: 'Send events to the Formo Events API'
openapi: 'POST /v0/raw_events'
---

Send events to Formo from any source. Sending custom events? See the [event specs](/data/events/track).

<Note>
  This endpoint is hosted on `events.formo.so`, a separate service from the main API at `api.formo.so`.
</Note>

<Note>
  Sending events from the backend? Use the [server-side SDK](https://docs.formo.so/sdks/server).
</Note>
