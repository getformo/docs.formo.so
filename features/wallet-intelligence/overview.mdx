## Features

Wallet intelligence helps you **understand and target users in web3**: 

- 🕵️‍♀️ <a href="/features/wallet-intelligence/wallet-profiles">**Wallet profiles.**</a> Turn anonymous wallets into high-value users with unified data.
- 🏃‍♀️‍➡️ <a href="/features/wallet-intelligence/audience-insights">**Audience insights.**</a> Know your onchain users with aggregated onchain insights.
- 🧑‍🧒‍🧒 <a href="/features/wallet-intelligence/segments">**User segments.**</a> Define and target users that share one or more profile properties.
- 🏷️ <a href="/features/wallet-intelligence/wallet-labels">**Wallet labels.**</a> Understand each user's preferences with autogenerated wallet labels.
- 💬 <a href="/features/wallet-intelligence/wallet-messaging">**Wallet messaging.**</a> Boost engagement with wallet-aware personalization.
- 📊 <a href="/features/wallet-intelligence/wallet-score">**Wallet reputation score.**</a> Use a scoring system to distinguish real users from bots.

Profile data is available for querying and export in the [Wallet Profile API](/api-reference/overview#wallet-profile-api).
